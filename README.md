🚀 Overview

BikeTool is a modern 10-Phase Mechanical Parts Booking System built using HTML5, CSS3, and Vanilla JavaScript.

The application provides an interactive multi-step booking workflow with animated mechanical gear visualization, intelligent part selection, invoice generation, and persistent progress using the browser's Local Storage.
